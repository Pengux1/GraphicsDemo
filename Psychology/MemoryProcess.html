Summary

Memory

Memory is not a single process or structure located in the brain. Instead, it is a collection of interconnnected and interacting functions and is represented throughout the brain by different mechanisms. Therefore instead of having a memory, we have memory systems. Although there are many different models for human memory, most involve three distinct processes:

Encoding

Conversion of information into a usable form so that it can be neurologically represented and stored in memory.

Encoding occurs first due to it occuring at the time of learning, and involves representing incoming sensory information into a usable form. How well it is encoded determines how well it will be stored, and retrieved.

Storage

Retention of the encoded information.

Retrieval

Recovery of stored information and bringing into conscious awareness for use when needed.

These three process occur in a sequence, interact and are interdependent.

Atkinson-Shiffrin's Multi-Store Model of Memory

The Atkinson–Shiffrin Multi-Store Model represents memory as consisting of three seperate stores called sensory memory, short-term memory and long-term memory. Each store processes information in different ways, and differs in function, capacity and duration of memory. The three stores operate simultaneously and interact in an integrated way. The model also describes memory in terms of it structural features and control processes.

Structural Features are the permanent, built in features of memory that do not vary from one situation to another. The three different stores are the basic structural features. Other structural features include the storage capacity of each store, and the storage duration of each store.

Control Processes are selected and used by each individual and may vary in different situations. They are under the concious control of the individual, and which one is used usually depends on what the individual does. These include: Attention, Rehersal and Retrival.

Attention

Whether or not an individual chooses to attend to and select incoming sensory information, determining whether that information is transferred from the sensory store to the short-term store.

Rehearsal

Determines whether information is retained in the short term store, how long it will be held there, and whether it is transferred to the long-term store.

Retrieval

Reffers to the specific retrieval method used by an individual, which determines whether some or all of the required information in the long-term store will be located, recoverd and brought into conscious awareness.

Sensory Memory

Sensory Memory is the entry point of memory where new incoming sensory information is stored for a brief period of time. The information received there is assumed to be retained as an exact copy of its original sensory form. Sensory memory is commonly described as having a potenitally unlimited storage capacity.

Sensory information remains in sensory memory just long enough for us to attend to and select the information to be transferred to short term memory for processing. Therefore it is a temporary storage system for information.

When we direct our attention to information in sensory information, this has the effect of transferring it to STM, where we become conciously aware of it. If sensory ifrmation is not attended to and no further processing occurs, it is permanently lost from experience. Incoming sensory information is assumed to be stored in seperate sensory systems, called sensory registers, each retaining a different sense, lasting for a different duration.

Iconic Memory

Iconic Memory is used to describe visual sensory memory. This store of memory tends to retain visual images in their original sensory form in iconic memory for about a third of a second.

Echoic Memory

Echoic Memory is used to desribe auditory sensory memory. This store of memory tends to retain sounds in their original sensory form in iconic memory for about 3 - 4 seconds. The longer duration for echoic memory is important for understanding speech, as words are a collection of sounds.

Sensory Memory has a large capacity, but a small duration.

Short-Term Memory (STM)

Short-Term Memory(STM) is a memory system with limited storage capactiy and limited duration, unless renewed. STM stores information temporarily, but for longer than sensory memory, and is sometimes reffered to as working memory due to information having to be in STM to be used. When attention is payed to information in sensory memeory or information retrieved from LTM, the information enters the STM. STM holds all information that you are consciously aware of at any moment in time.

Duration of STM

In general, information is stored well for the first few seconds, but after around 12 seconds, recall starts to decline, and by 18 seconds almost of the information disappears entirely if it has not been renewed. Information can linger in STM for up to 30 seconds, so its duration is described as up to 30 seconds". Distraction can prevent rehearsal, resulting in the loss of the information, and new information acquired when distracted may exceed the limited capactiy of STM and dosplace it from STM.

Capactiy of STM

Compared to Sensory Memory and LTM, STM has a very limited storage capactiy. The amount of information that STM can hold at any time is desribed as 7 ± 2 bits of information. When STM is full, new items can onlt be added by displacing old items. Space is also filled when thinking, and when information is tmeporarily retrieved from LTM to be used or updated.

Information stored in STM is lost primarily through decay and displacement. Decay of information in STM occurs when information is not renewd, and fades over time.

Chunking

Chunking is a method to get around the limited capacity of STM. Information is 'combined' into chunks, not exceeding 7 ± 2 bits all together. By doing this, the overall bits of information needed to be stored is reduced, and the amount of information retained by STM is increased.

Long Term Memory (LTM)

Long-Term Memory stores a potentially unlimited amount of information for a long period of time(Possibly permanently), and is therefore described as long in duration and large in capacity. LTM is not a single store for all information, with different types of memory being stored in different types of LTM. The two main types of LTM are Explicit and Implicit Memory.

Explicit Memory

Explicit Memory involves memory that occurs when information can be conciously and intentionally retreived. Explicit Memories are also called declarative memories because they can be consciously retreived, and can be openly state the memory. The most common tests of explicit memory involve recall and recognition, which require conscious retrieval of specific memories. Explicit memory has two sub-types, Episodic and Semantic memory.

Episodic Memory

Episodic Memory is the memory of personally experienced events. These memories often include details of the time, place and psychological and phisological state when the event occurs. Episodic Memory makes it possible to be consciously aware of an earlier experience in a certain situation at a certain time.
